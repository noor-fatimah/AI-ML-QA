# Machine Learning Basics

## What is Machine Learning?
Machine Learning is a branch of AI where systems learn patterns from data instead of being explicitly programmed.


## Types of Machine Learning

### 1. Supervised Learning
- Uses labeled data
- Has input and correct output
- Learns mapping function

Examples:
- Spam detection
- House price prediction

#### Types:
- Regression → Predict continuous values
- Classification → Predict categories

---

### 2. Unsupervised Learning
- Uses unlabeled data
- Finds hidden patterns
- No correct output provided

Examples:
- Customer segmentation
- Clustering

---

## Regression
- Predicts numerical value
- Example: Predicting salary, temperature

---

## Classification
- Predicts category/label
- Example: Spam or Not Spam
- Binary Classification → 2 classes
- Multi-class Classification → More than 2 classes
